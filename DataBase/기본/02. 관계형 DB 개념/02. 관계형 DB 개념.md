### 수학에서 relation

![alt text](<수학에서 관계.drawio.svg>)

relation은 튜플의 집합이다.

튜플은 릴레이션에서 각 Set간 연결된 하나의 리스트를 의미한다.

이러한 용어는 relation 모델에서도 동일하게 사용된다.

#### SQL 용어 비교

| relation data model |  SQL   |
| :-----------------: | :----: |
|      relation       | table  |
|      attribute      | column |
|        tuple        |  row   |
|       domain        | domain |
